## Lista Ligada (Linked List)

 A Lista Ligada (Linked List), também chamada de Lista Simplesmente Ligada, é uma estrutura de dados dinâmica formada por várias células conhecidas como nós. Cada nó contém um conteúdo (dado) e uma referência (ponteiro) para o próximo nó da sequência.Esse mecanismo é conhecido como encadeamento, pois os nós ficam conectados entre si por meio dos ponteiros, formando uma sequência lógica de elementos, mesmo estando localizados em diferentes locais da memorias , como se realmente fossem elos de correntes ligados.Desse modo , essa característica permite realizar remoções e inserções em diferentes posições da lista de forma prática e flexível.  
    Além disso, por ser uma estrutura dinâmica, diferente de estruturas estáticas como matrizes e vetores, a lista ligada não exige que os elementos estejam armazenados em áreas contíguas da memória. Isso possibilita um alocamento de memória mais eficiente, aproveitando melhor os recursos disponíveis do sistema.

Métodos comentados:

- inserirNoFim()

- removerPorPosicao()

- tamanhoDaLista()
